polished

A lightweight toolset for writing styles in JavaScript

View on GitHub Docs

Try editing the code below

		
const styles = {
  color: lighten('#000', 20),
  fontSize: modularScale(1),
  [hiDPI(1.5)]: {
    fontSize: modularScale(1.25)
  }
}
		
const styles = {
  color: '#333',
  fontSize: '16px',
  '@media only screen and': {
    fontSize: '20px'
  }
}
