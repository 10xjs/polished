stripUnit

src/helpers/stripUnit.js

Strip the unit from a given CSS value

stripUnit(value: string): Number
Parameters
value (string) A string containing a value with its unit.
Returns
Number: Unitless value.
Example
// Styles as object usage
const styles = {
  '--dimension': stripUnit(100px)
}

// styled-components usage
const div = styled.div`
  --dimension: ${stripUnit(100px)}
`

// CSS in JS Output

element {
  '--dimension': 100
}

ellipsis

src/mixins/ellipsis.js

CSS to represent truncated text with an ellipsis.

ellipsis(width: string?): Object
Parameters
width (string? = 100%) Max-width for text to respect before being truncated.
Returns
Object: CSS as JS.
Example
// Styles as object usage
const styles = {
  ...ellipsis(250px)
}

// styled-components usage
const div = styled.div`
  ${ellipsis(250px)}
`

// CSS as JS Output

div {
  'display': 'inline-block',
  'max-width': '250px',
  'overflow': 'hidden',
  'text-overflow': 'ellipsis',
  'white-space': 'nowrap',
  'word-wrap': 'normal'
}
